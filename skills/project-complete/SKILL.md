# Project Complete - End-of-Project Documentation

## Purpose
Generate comprehensive documentation suite at project/sprint completion for content marketing, social media, and stakeholder communication. Creates detailed build journals, feature documentation, and marketing materials.

## When to Invoke
- "Project complete - create documentation"
- "Generate marketing materials for this build"
- "Create content for [product] launch"
- "Document this sprint for content"
- End of 5-Day Sprint
- Product launch preparation

---

## Documents to Generate

Create ALL of the following as separate markdown files in `docs/public/`:

### 1. Build Journal (5-Day Chronicle)
**File**: `docs/public/build-journal.md`

Chronological, detailed account of the entire build process. This becomes source material for blog posts, Twitter threads, and LinkedIn content.
